## 1. 平台概述

### 1.1 平台简介

UI 自动化测试平台是一款企业级自动化测试工具，帮助测试团队高效管理测试用例、执行测试任务并生成测试报告。

### 1.2 系统要求

| 项目 | 要求 |
|------|------|
| 浏览器 | Chrome 90+ / Firefox 88+ / Edge 90+ |
| 分辨率 | 1280 x 720 或更高 |
| 网络 | 可访问服务器 10.254.239.205:8080 |

### 1.3 访问地址

- 🌐 **平台地址：** http://10.254.239.205（经nginx代理）
- 🔌 **API 地址：** http://10.254.239.205:8888

### 1.4 主要功能模块

| 模块 | 功能描述 |
|------|----------|
| 测试用例 | 创建、编辑、删除测试用例及步骤 |
| 执行记录 | 执行测试用例，查看执行结果和步骤详情，AI token消耗统计 |
| 测试任务 | 定时任务管理，支持CRON表达式配置 |
| 用户管理 | 添加、编辑、删除用户（15条/页分页） |
| 部门管理 | 管理部门组织结构 |

## 3. 功能使用指南

### 3.1 首页

首页展示平台概览信息：

- **统计数据** - 用例数、任务数、报告数
- **最近执行** - 最近的测试执行记录
- **快捷操作** - 快速创建任务、查看报告

### 3.2 用户管理

#### 查看用户信息

1. 点击左侧菜单「用户管理」
2. 查看当前用户列表（默认15条/页）
3. 支持分页浏览

#### 添加用户

1. 在用户管理页面，点击「添加用户」按钮
2. 填写用户名、昵称、邮箱、密码
3. 设置用户状态（正常/禁用）
4. 点击「确定」保存

#### 编辑用户

1. 在用户列表中，点击对应行的「编辑」按钮
2. 修改用户信息（用户名不可修改）
3. 点击「确定」保存修改

#### 删除用户

1. 在用户列表中，点击对应行的「删除」按钮
2. 确认删除操作（超级管理员admin不可删除）
3. 用户将被从系统中移除

#### 修改个人信息

1. 点击右上角用户名
2. 选择「个人设置」
3. 修改昵称、邮箱等信息
4. 点击「保存」

### 3.3 测试用例管理

#### 创建测试用例

1. 点击左侧菜单「测试用例」
2. 点击「新建用例」按钮，跳转至用例详情页
3. 填写用例信息：
   - 用例编号（自动生成，格式：TCyyyymmdd+4位序号）
   - 用例名称（必填）
   - 用例描述
   - 设计者（必填）
   - 用例性质：正例/反例
   - 状态：启用/禁用
4. 点击「保存」

#### 用例步骤管理

保存用例基本信息后，下方会显示「用例步骤」区域：

1. 点击「新增步骤」添加步骤
2. 填写步骤信息：
   - 步骤描述（必填）
   - 断言描述
   - 是否启用（默认启用）
   - 是否截图（默认截图）
3. 点击「保存」
4. 支持批量选择删除步骤
5. 支持 Excel 导入/导出步骤

#### 用例导入导出

1. 用例列表支持 Excel 导入/导出
2. 点击「导入」选择 Excel 文件（.xlsx/.xls）
3. 点击「导出」下载用例数据

#### 批量删除

1. 勾选需要删除的用例
2. 点击「批量删除」按钮
3. 确认删除操作

#### 查看执行记录

1. 在用例列表中，点击目标用例右侧的「操作」按钮
2. 选择「执行记录」
3. 在弹出的执行记录对话框中，上半部分显示该用例的历史执行记录（默认5条/页）
4. 点击任意一条执行记录，下方会自动加载并显示该次执行的步骤记录
5. 可查看每个步骤的执行状态、耗时、断言状态、截图和AI token消耗
6. **AI token消耗说明**：显示问号图标，hover显示估算规则：中文每2字符≈1 token，英文每4字符≈1 token

#### 执行记录自动刷新

1. 在执行记录页面，上方列表和下方步骤列表均支持自动刷新
2. 点击列表右上角的「开启自动刷新」按钮启用每秒自动刷新
3. 刷新开启后，按钮变为「停止刷新」，点击可停止自动刷新
4. 自动刷新标签会显示在列表标题旁

#### 编辑/删除测试用例

1. 在用例列表中找到目标用例
2. 点击「编辑」按钮，跳转至用例详情页修改
3. 或点击「删除」按钮确认删除

### 3.4 测试任务

#### 创建测试任务

1. 点击左侧菜单「测试任务」→「测试任务」
2. 点击「新建任务」按钮
3. 填写任务信息：
   - 任务名称（必填）
   - 任务编号（自动生成）
   - Cron表达式（定时任务配置）
   - 任务配置（JSON格式）
   - 备注
4. 点击「保存」

#### 测试任务管理

1. **查看任务列表**：显示所有任务，支持分页
2. **编辑任务**：点击「编辑」按钮修改任务信息
3. **删除任务**：点击「删除」按钮删除任务
4. **启用/禁用任务**：通过状态开关控制任务是否启用

### 3.5 执行记录

#### 查看执行记录

1. 点击左侧菜单「测试任务」→「执行记录」
2. 查看所有测试用例的执行记录
3. 点击任意一条执行记录，下方显示该次执行的步骤详情
4. 可查看执行状态、耗时、AI token消耗等信息
5. **AI token消耗说明**：显示问号图标，hover显示估算规则：中文每2字符≈1 token，英文每4字符≈1 token

### 3.5 测试报告

#### 查看报告

1. 点击左侧菜单「测试报告」
2. 选择时间范围或筛选条件
3. 查看报告列表
4. 点击报告查看详情

#### 导出报告

1. 打开目标报告
2. 点击「导出」按钮
3. 选择导出格式（PDF/HTML）
4. 下载报告文件

## 4. 常见问题

### Q1: 登录提示「用户名或密码错误」

**解决方案：**
- 检查用户名是否正确（默认：admin）
- 检查密码是否正确（默认：admin123）
- 如忘记密码，请联系管理员重置

### Q2: 页面显示异常

**解决方案：**
- 清除浏览器缓存：Ctrl + Shift + Delete
- 刷新页面：F5 或 Ctrl + R
- 建议使用 Chrome 90+ 浏览器

### Q3: 登录后提示「网络错误」

**解决方案：**
- 检查网络是否可访问服务器
- 确认服务器服务是否正常运行
- 尝试刷新页面

### Q4: 无法打开页面（ERR_UNSAFE_PORT）

**解决方案：**
- 这是 Chrome 浏览器的安全限制
- 使用 Firefox 或 Edge 浏览器访问
- 或联系管理员更换端口

### Q5: 测试执行失败

**解决方案：**
1. 检查网络连接是否正常
2. 查看失败原因描述
3. 如是元素定位问题，检查页面是否变化
4. 如是网络问题，重试执行

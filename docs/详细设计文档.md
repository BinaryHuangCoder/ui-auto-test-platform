# UI 自动化测试平台 - 详细设计文档

## 1. 系统架构设计

### 1.1 整体架构
系统采用前后端分离架构，分为四层：
- **表现层**：Vue 3 前端应用，提供用户交互界面
- **业务层**：Spring Boot 后端服务，处理业务逻辑
- **数据层**：MySQL 数据库，存储测试用例和执行记录
- **执行层**：Node.js + Playwright + Midscene，执行测试用例

### 1.2 模块划分

## 2. 数据库设计

### 2.1 测试用例表 (test_case)
```sql
CREATE TABLE test_case (
    id BIGINT PRIMARY KEY AUTO_INCREMENT,
    name VARCHAR(200) NOT NULL COMMENT '用例名称',
    description TEXT COMMENT '用例描述',
    category VARCHAR(50) COMMENT '分类',
    status TINYINT DEFAULT 1 COMMENT '状态：1-启用 0-禁用',
    created_by VARCHAR(50) COMMENT '创建人',
    created_time DATETIME DEFAULT CURRENT_TIMESTAMP,
    updated_time DATETIME DEFAULT CURRENT_TIMESTAMP ON UPDATE CURRENT_TIMESTAMP
);
```

### 2.2 测试步骤表 (test_case_step)
```sql
CREATE TABLE test_case_step (
    id BIGINT PRIMARY KEY AUTO_INCREMENT,
    case_id BIGINT NOT NULL COMMENT '用例 ID',
    step_no INT NOT NULL COMMENT '步骤序号',
    step_type VARCHAR(20) COMMENT '步骤类型：open/click/type/wait/ai-query',
    step_description TEXT COMMENT '步骤描述',
    target VARCHAR(500) COMMENT '目标元素/URL',
    value VARCHAR(500) COMMENT '输入值',
    assertion_description TEXT COMMENT '断言描述',
    screenshot TINYINT DEFAULT 0 COMMENT '是否截图：1-是 0-否',
    created_time DATETIME DEFAULT CURRENT_TIMESTAMP
);
```

    step_description TEXT COMMENT '步骤描述',

## 4. 核心流程设计

### 4.1 测试执行流程
```
1. 用户触发执行 → 2. 创建执行记录 → 3. 加载测试步骤
       ↓
4. 初始化浏览器 → 5. 循环执行步骤 → 6. 捕获截图
       ↓
7. 执行 AI 断言 → 8. 保存执行结果 → 9. 关闭浏览器
       ↓
10. 更新执行状态 → 11. 返回执行结果
```

### 4.2 截图处理流程
```
1. 步骤执行完成 → 2. Playwright 截图 → 3. 转 Base64 编码
       ↓
4. 返回 JSON 结果 → 5. 后端提取 Base64 → 6. 保存到数据库
       ↓
7. 前端展示截图
```

## 5. 关键代码设计

### 5.1 执行器核心逻辑 (executor.js)
```javascript
async function executeStep(action, target, value) {
  const result = { success: false, screenshot: null, screenshotBase64: null };
  
  // 执行动作
  switch (action) {
    case 'open': await page.goto(target); break;
    case 'click': await page.click(target); break;
    case 'type': await page.fill(target, value); break;
    case 'ai-query': const queryResult = await agent.ai(target); break;
  }
  
  // 截图
  const screenshot = await page.screenshot();
  result.screenshotBase64 = 'data:image/png;base64,' + screenshot.toString('base64');
  
  return result;
}
```

### 5.2 后端截图提取逻辑
```java
private String extractScreenshot(String result) {
    // 从 screenshotBase64 字段获取（长度 20 字符）
    int base64Start = result.indexOf("\"screenshotBase64\":\"");
    if (base64Start >= 0) {
        base64Start += 20;  // 跳过字段名
        int base64End = result.indexOf("\"", base64Start);
        return result.substring(base64Start, base64End);
    }
    return null;
}
```
